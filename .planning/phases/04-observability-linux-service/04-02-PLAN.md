  <action>
Create the directory deploy/systemd/ and the file deploy/systemd/cee-exporter.service with this exact content:

[Unit]
Description=Dell PowerStore CEPA to Windows Event Log bridge
Documentation=https://github.com/fjacquet/cee-exporter
After=network-online.target
Wants=network-online.target

[Service]
Type=simple
User=cee-exporter
Group=cee-exporter
WorkingDirectory=/etc/cee-exporter
ExecStart=/usr/local/bin/cee-exporter -config /etc/cee-exporter/config.toml
Restart=on-failure
RestartSec=5s
TimeoutStopSec=30s
Environment=CEE_LOG_FORMAT=json
EnvironmentFile=-/etc/cee-exporter/env

# File descriptor limit for high-throughput CEPA ingestion
LimitNOFILE=65536

# Filesystem hardening — daemon only needs write access to its log dir
ProtectSystem=strict
ReadWritePaths=/var/log/cee-exporter
PrivateTmp=true
NoNewPrivileges=true

[Install]
WantedBy=multi-user.target

Critical field explanations (DO NOT deviate):
- Type=simple: Go daemons do not fork. Type=forking would cause systemd to wait indefinitely.
  Type=notify requires sd_notify integration (deferred to OBS-F02).
- After=network-online.target: The daemon connects to GELF/syslog targets at startup.
  network.target only waits for the network stack to initialize, not for IP addresses to be
  assigned. network-online.target waits for routable IPs. Use Wants= (not Requires=) to
  avoid hard boot failure if networkd is not installed.
- Restart=on-failure + RestartSec=5s: Auto-restart after crash. RestartSec=5s prevents a
  tight restart loop if the binary crashes immediately (e.g. bad config).
- ProtectSystem=strict: Mounts filesystem read-only except explicitly listed paths.
  ReadWritePaths=/var/log/cee-exporter allows log writes. Without ReadWritePaths, any write
  by the daemon fails with "read-only file system".
- EnvironmentFile=-/etc/cee-exporter/env: The leading dash means systemd ignores missing
  file (not an error). Operators can place env overrides there.
- User=cee-exporter: Operator must create this user before enabling. Add a comment to the
  file or Makefile output directing the operator to run:
    useradd --system --no-create-home --shell /usr/sbin/nologin cee-exporter
  </action>
