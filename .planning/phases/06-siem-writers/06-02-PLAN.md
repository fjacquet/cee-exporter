<task type="tdd">
  <name>Task 2: GREEN — Implement BeatsWriter to pass all tests</name>
  <files>pkg/evtx/writer_beats.go</files>
  <action>
Create pkg/evtx/writer_beats.go with package evtx. Follow the GELFWriter pattern from pkg/evtx/writer_gelf.go exactly.

CRITICAL: SyncClient is NOT thread-safe — every Send call MUST be inside w.mu.Lock().
CRITICAL: SyncClient cannot be reused after error — on send failure, close and recreate via w.dial().
CRITICAL: go-lumber has NO built-in TLS Option — inject TLS via SyncDialWith + tls.Dialer custom function.

File header comment:
```go
// BeatsWriter — all platforms.
// Sends Lumberjack v2 frames to Logstash or Graylog Beats Input.
// TLS: injected via SyncDialWith with tls.Dialer (go-lumber has no TLS Option).
// Thread safety: sync.Mutex serialises SyncClient.Send (SyncClient is not thread-safe).
// Reconnect: SyncClient closed and recreated on send failure (SyncClient cannot recover).
package evtx
```

Implement:

1. BeatsConfig struct:
   - Host string
   - Port int (default 5044 in constructor)
   - TLS bool

2. BeatsWriter struct:
   - cfg BeatsConfig
   - mu sync.Mutex
   - client *lumberv2.SyncClient

3. NewBeatsWriter(cfg BeatsConfig) (*BeatsWriter, error):
   - Set default Port = 5044 if zero
   - Call w.dial(); return error if fails
   - Log slog.Info("beats_writer_ready", "host", "port", "tls")
   - Return writer

4. dial() error:
   - addr := net.JoinHostPort(w.cfg.Host, fmt.Sprintf("%d", w.cfg.Port))
   - If TLS: create tls.Dialer{NetDialer: &net.Dialer{Timeout: 5*time.Second}, Config: &tls.Config{MinVersion: tls.VersionTLS12}}
     Call lumberv2.SyncDialWith(func(network, address string) (net.Conn, error) { return tlsDialer.DialContext(context.Background(), network, address) }, addr, lumberv2.Timeout(30*time.Second))
   - If not TLS: lumberv2.SyncDial(addr, lumberv2.Timeout(30*time.Second))
   - Assign w.client = cl

5. WriteEvent(ctx context.Context, e WindowsEvent) error:
   - Build event: event := buildBeatsEvent(e)
   - Lock w.mu
   - _, err := w.client.Send([]interface{}{event})
   - On error: log slog.Warn("beats_reconnect", "reason", err); close w.client; call w.dial(); retry w.client.Send once
   - Log slog.Debug("beats_event_sent", "event_id", e.EventID, "cepa_event_type", e.CEPAEventType)

6. Close() error:
   - Lock w.mu, call w.client.Close() if non-nil

7. buildBeatsEvent(e WindowsEvent) map[string]interface{}:
   - Return map with:
     "@timestamp":      e.TimeCreated.UTC().Format(time.RFC3339Nano)
     "message":         fmt.Sprintf("%s on %s", e.CEPAEventType, e.ObjectName)
     "event_id":        e.EventID
     "provider":        e.ProviderName
     "computer":        e.Computer
     "user":            e.SubjectUsername
     "domain":          e.SubjectDomain
     "user_sid":        e.SubjectUserSID
     "logon_id":        e.SubjectLogonID
     "object_name":     e.ObjectName
     "object_type":     e.ObjectType
     "access_mask":     e.AccessMask
     "accesses":        e.Accesses
     "client_address":  e.ClientAddr
     "cepa_event_type": e.CEPAEventType

Import alias: lumberv2 "github.com/elastic/go-lumber/client/v2"
Other imports: context, crypto/tls, fmt, log/slog, net, sync, time
