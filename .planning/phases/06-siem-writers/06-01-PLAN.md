  <name>Task 2: GREEN — Implement SyslogWriter to pass all tests</name>
  <files>pkg/evtx/writer_syslog.go</files>
  <action>
Create pkg/evtx/writer_syslog.go with package evtx. Follow the GELFWriter pattern from pkg/evtx/writer_gelf.go exactly — same struct layout, same mutex pattern, same reconnect-once logic.

File structure (no build tags — crewjam/rfc5424 and net are cross-platform):

```go
// SyslogWriter — all platforms.
// Sends RFC 5424 messages to a syslog receiver over UDP or TCP.
// UDP: single datagram, no framing.
// TCP: RFC 6587 §3.4.1 octet-counting framing.
// Uses github.com/crewjam/rfc5424 for message construction (cross-platform,
// unlike stdlib log/syslog which excludes Windows).
package evtx
```

Implement:

1. SyslogConfig struct:
   - Host string
   - Port int (default 514 in constructor)
   - Protocol string (default "udp" in constructor)
   - AppName string (default "cee-exporter" in constructor)

2. SyslogWriter struct:
   - cfg SyslogConfig
   - mu sync.Mutex
   - conn net.Conn

3. NewSyslogWriter(cfg SyslogConfig) (*SyslogWriter, error):
   - Set defaults for Port (514), Protocol ("udp"), AppName ("cee-exporter") if zero
   - Call w.connect(); return error if fails
   - Log slog.Info("syslog_writer_ready", "host", "port", "protocol")
   - Return writer

4. connect() error:
   - addr := net.JoinHostPort(w.cfg.Host, fmt.Sprintf("%d", w.cfg.Port))
   - conn, err := net.DialTimeout(w.cfg.Protocol, addr, 5*time.Second)
   - Assign w.conn = conn

5. send(payload []byte) error:
   - TCP case: fmt.Fprintf(w.conn, "%d ", len(payload)) then w.conn.Write(payload)
   - UDP (default) case: w.conn.Write(payload)

6. WriteEvent(ctx context.Context, e WindowsEvent) error:
   - Call buildSyslog5424(e, w.cfg.AppName) to get payload
   - Lock mutex
   - Call w.send(payload); on error: log slog.Warn("syslog_reconnect"), reconnect, retry once
   - Log slog.Debug("syslog_event_sent", "event_id", e.EventID)

7. Close() error:
   - Lock mutex, close w.conn if non-nil

8. buildSyslog5424(e WindowsEvent, appName string) ([]byte, error):
   - Create rfc5424.Message with:
     - Priority: rfc5424.Daemon | rfc5424.Info
     - Timestamp: e.TimeCreated
     - Hostname: e.Computer
     - AppName: appName
     - ProcessID: fmt.Sprintf("%d", e.ProcessID)
     - MessageID: fmt.Sprintf("%d", e.EventID)
     - Message: []byte(fmt.Sprintf("%s on %s", e.CEPAEventType, e.ObjectName))
   - sdID := "audit@32473"
   - m.AddDatum(sdID, "EventID", fmt.Sprintf("%d", e.EventID))
   - m.AddDatum(sdID, "User", e.SubjectUsername)
   - m.AddDatum(sdID, "Domain", e.SubjectDomain)
   - m.AddDatum(sdID, "Object", e.ObjectName)
   - m.AddDatum(sdID, "AccessMask", e.AccessMask)
   - m.AddDatum(sdID, "ClientAddr", e.ClientAddr)
   - m.AddDatum(sdID, "CEPAType", e.CEPAEventType)
   - return m.MarshalBinary()

Imports: context, fmt, log/slog, net, sync, time, github.com/crewjam/rfc5424
