PART A — Create cmd/cee-exporter/service_notwindows.go:

The file must have exactly this content:

  //go:build !windows

  package main

  // runWithServiceManager runs the program directly on non-Windows platforms.
  // On Windows, this function is replaced by service_windows.go (Phase 5),
  // which wraps run() with the Windows Service Control Manager.
  func runWithServiceManager(runFn func()) {
      runFn()
  }

The build constraint //go:build !windows ensures this file compiles on Linux and macOS but
NOT on Windows. Phase 5 will add service_windows.go with a different implementation of
runWithServiceManager for the Windows Service wrapper.

Do NOT add a blank line between the build constraint and the package declaration — the
//go:build comment must be on the first line of the file (or after the copyright header
if one is added, but no copyright header is needed here).
