<objective>
Implement the pure binary format helpers for the EVTX file format: FILETIME encoding, UTF-16LE string encoding, file header construction, chunk header construction with CRC32 deferred-patch, and event record wrapping.

Purpose: These helpers are the low-level foundation that the BinaryEvtxWriter (Plan 02) consumes. Isolating them here allows thorough unit testing with pure math verification before any Writer interface work begins. This is the most failure-prone layer — wrong byte offsets or CRC scope produce silently corrupt files.

Output: pkg/evtx/evtx_binformat.go (helpers) and pkg/evtx/evtx_binformat_test.go (unit tests). No build tag — helpers work on all platforms, enabling Linux CI coverage.

Create pkg/evtx/evtx_binformat.go in package evtx. NO build tag — these helpers are platform-agnostic math and are needed by the writer only through the !windows-tagged file.

Implement the following exported helpers using stdlib only (encoding/binary, hash/crc32, unicode/utf16):

**Constants:**

```go
const (
    evtxFileMagic       = "ElfFile\x00"  // 8 bytes
    evtxChunkMagic      = "ElfChnk\x00"  // 8 bytes
    evtxRecordSignature = uint32(0x00002A2A)
    evtxFileHeaderSize  = 4096
    evtxChunkSize       = 65536
    evtxChunkHeaderSize = 512
    filetimeEpochDelta  = int64(116444736000000000) // 100ns intervals
)
```

**toFILETIME(t time.Time) uint64**
Convert Go time.Time to Windows FILETIME (100ns intervals since 1601-01-01 UTC).
Formula: uint64(t.UTC().UnixNano()/100 + filetimeEpochDelta)

**encodeUTF16LE(s string) []byte**
Encode a Go string as UTF-16LE with null terminator, prefixed by uint16 character count.
Layout: [uint16 char_count][uint16 codeunit_0][uint16 codeunit_N][0x0000]
Use unicode/utf16.Encode to get []uint16, then write each via binary.LittleEndian.PutUint16.
Total byte count: 2 + len(u16)*2 + 2

**buildFileHeader(chunkCount uint16, nextRecordID uint64) []byte**
Returns 4096-byte slice. Field layout (all little-endian):

- [0:8]   Signature "ElfFile\x00"
- [8:16]  FirstChunkNumber = 0
- [16:24] LastChunkNumber = uint64(chunkCount-1)
- [24:32] NextRecordIdentifier = nextRecordID
- [32:36] HeaderSize = 128
- [36:38] MinorVersion = 1
- [38:40] MajorVersion = 3
- [40:42] BlockSize = 4096
- [42:44] ChunkCount = chunkCount
- [44:120] reserved zeros
- [120:124] Flags = 0
- [124:128] CRC32 (see below)
- [128:4096] padding zeros

CRC32: compute crc32.Checksum(buf[0:120], crc32.IEEETable) with buf[124:128] still zero, then write the result at buf[124:128].

**buildChunkHeader(firstRecordID, lastRecordID uint64, recordCount uint16, freeSpaceOffset uint32) []byte**
Returns 512-byte slice. Field layout:

- [0:8]   Signature "ElfChnk\x00"
- [8:16]  FirstEventRecordNumber = firstRecordID
- [16:24] LastEventRecordNumber = lastRecordID
- [24:32] FirstEventRecordIdentifier = firstRecordID
- [32:40] LastEventRecordIdentifier = lastRecordID
- [40:44] HeaderSize = 128
- [44:48] LastEventRecordDataOffset = freeSpaceOffset (relative to chunk start)
- [48:52] FreeSpaceOffset = freeSpaceOffset
- [52:56] EventRecordsCRC32 = 0 (caller patches after assembling records; set to 0 here)
- [56:120] reserved zeros
- [120:124] Flags = 0
- [124:128] HeaderCRC32 = 0 (caller must call patchChunkCRC after)
- [128:512] string table / reserved zeros

**patchChunkCRC(chunk []byte)**
Compute the chunk header CRC32 over bytes [0:120] and [128:512] (skip bytes [120:128] per spec).
Zero bytes [120:128] first, then:
  h := crc32.New(crc32.IEEETable)
  h.Write(chunk[0:120])
  h.Write(chunk[128:512])
Write h.Sum32() at chunk[124:128].

**patchEventRecordsCRC(chunk []byte, recordsStart, recordsEnd int)**
Compute CRC32 of chunk[recordsStart:recordsEnd] and write at chunk[52:56].
Use crc32.Checksum(chunk[recordsStart:recordsEnd], crc32.IEEETable).

**wrapEventRecord(recordID uint64, timestamp uint64, binXMLPayload []byte) []byte**
Record layout:

- [0:4]   Signature = 0x00002A2A (little-endian)
- [4:8]   Size = uint32(24 + len(binXMLPayload) + 4)
- [8:16]  EventRecordID = recordID
- [16:24] TimeCreated = timestamp (FILETIME)
- [24:24+len(payload)] BinXML payload
- [end-4:end] Size copy (same value as offset 4)

Return the complete record as []byte.

Imports: encoding/binary, hash/crc32, time, unicode/utf16

Create pkg/evtx/evtx_binformat_test.go in package evtx (white-box). No build tag — tests run on Linux CI.

Use stdlib only (testing, encoding/binary, hash/crc32, time). NO testify. Table-driven with t.Run.

Implement the following test functions:

**TestToFILETIME**
Table cases:

- Unix epoch (1970-01-01 00:00:00 UTC) → expect 116444736000000000
- A known timestamp: 2024-01-01 00:00:00 UTC → compute expected manually
  Expected = (1704067200 * 10_000_000) + 116444736000000000
  (1704067200 is seconds since Unix epoch for 2024-01-01)
- Verify result is > 0 and < math.MaxUint64

**TestEncodeUTF16LE**
Table cases:

- Empty string "" → [0x00,0x00, 0x00,0x00] (char_count=0, null terminator only)
- "A" → [0x01,0x00, 0x41,0x00, 0x00,0x00] (count=1, 'A' in UTF-16LE, null)
- "AB" → [0x02,0x00, 0x41,0x00, 0x42,0x00, 0x00,0x00]
For each: check len(result) == 2 + len(runes)*2 + 2; check first 2 bytes are char count; check last 2 bytes are 0x00,0x00.

**TestBuildFileHeader**

- Call buildFileHeader(1, 42)
- Verify len(result) == 4096
- Verify result[0:8] == []byte("ElfFile\x00")
- Verify binary.LittleEndian.Uint16(result[36:]) == 1 (minor version)
- Verify binary.LittleEndian.Uint16(result[38:]) == 3 (major version)
- Verify binary.LittleEndian.Uint16(result[40:]) == 4096 (block size)
- Verify binary.LittleEndian.Uint16(result[42:]) == 1 (chunk count)
- Verify binary.LittleEndian.Uint64(result[24:]) == 42 (next record ID)
- Verify CRC32: zero out result[124:128] in a copy, compute crc32.Checksum(copy[0:120], crc32.IEEETable), compare to binary.LittleEndian.Uint32(result[124:])

**TestWrapEventRecord**

- Call wrapEventRecord(1, 12345678, []byte{0xAA, 0xBB})
- Verify result[0:4] == []byte{0x2A, 0x2A, 0x00, 0x00} (signature)
- Verify size = uint32(24 + 2 + 4) = 30; check result[4:8] and result[26:30] both == 30
- Verify binary.LittleEndian.Uint64(result[8:]) == 1 (record ID)
- Verify binary.LittleEndian.Uint64(result[16:]) == 12345678 (timestamp)
- Verify result[24:26] == []byte{0xAA, 0xBB} (payload)

**TestPatchChunkCRC**

- Create a 512-byte zeroed slice
- Call patchChunkCRC on it
- Verify result[124:128] is NOT all zeros (CRC was written)
- Recompute: h := crc32.New(crc32.IEEETable); h.Write(slice[0:120]); h.Write(slice[128:512])
- Verify binary.LittleEndian.Uint32(slice[124:]) == h.Sum32()
