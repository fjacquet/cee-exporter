**Struct definition:**

```go
type BinaryEvtxWriter struct {
    mu        sync.Mutex
    path      string     // output file path (not a directory)
    records   []byte     // accumulated event records for current chunk
    recordID  uint64     // monotonically incrementing, starts at 1
    firstID   uint64     // first record ID in current chunk
}
```

**NewBinaryEvtxWriter(evtxPath string) (*BinaryEvtxWriter, error)**

- evtxPath is the output file path (e.g., "/var/log/cee-exporter.evtx"), not a directory
- Validate evtxPath is non-empty; return fmt.Errorf if empty
- Create parent directory if it does not exist using os.MkdirAll with path.Dir(evtxPath)
- Initialize recordID = 1, firstID = 1
- Log slog.Info("binary_evtx_writer_ready", "path", evtxPath)
- Return writer, nil

**WriteEvent(ctx context.Context, e WindowsEvent) error**

- Lock mu; defer unlock
- Build BinXML payload using buildBinXML(e) (see below)
- Compute FILETIME: ts := toFILETIME(e.TimeCreated)
- Build event record: rec := wrapEventRecord(w.recordID, ts, payload)
- Append to w.records: w.records = append(w.records, rec...)
- Increment w.recordID
- If len(w.records) approaches evtxChunkSize-evtxChunkHeaderSize (say > 60000 bytes), flush chunk and start new one (call w.flushChunk internally; reset w.records, w.firstID = w.recordID)
- Log slog.Debug("binary_evtx_event_buffered", "event_id", e.EventID, "record_id", w.recordID-1)
- Return nil

**Close() error**

- Lock mu; defer unlock
- If len(w.records) == 0: log warning "binary_evtx_writer_closed_empty" and return nil
- Call w.flushToFile()
- Return any error from flushToFile

**flushToFile() error (unexported)**
Called from Close() while holding the mutex. Assembles and writes the complete file:

1. Build chunk header: buildChunkHeader(w.firstID, w.recordID-1, uint16(number_of_records), uint32(evtxChunkHeaderSize+len(w.records)))
2. Patch event records CRC: patchEventRecordsCRC(chunkHeader, evtxChunkHeaderSize, evtxChunkHeaderSize+len(w.records)) — but since chunkHeader is only 512 bytes, compute CRC separately: crc32.Checksum(w.records, crc32.IEEETable) and write at chunkHeader[52:56]
3. Combine chunk: chunkBytes := make([]byte, evtxChunkSize); copy(chunkBytes[0:], chunkHeader); copy(chunkBytes[evtxChunkHeaderSize:], w.records); zero-pad the rest
4. Patch chunk header CRC: patchChunkCRC(chunkBytes)
5. Build file header: buildFileHeader(1, w.recordID)
6. Write to file atomically: os.WriteFile(w.path, append(fileHeader, chunkBytes...), 0o644)
7. Log slog.Info("binary_evtx_file_written", "path", w.path, "records", w.recordID-w.firstID)

**buildBinXML(e WindowsEvent) []byte (unexported)**
This is the static template approach. Encode the Event XML structure as BinXML tokens.

The BinXML encoding for a Windows Security event uses these token types (from libevtx spec):

- 0x0F = BinXmlFragmentHeader token (must open the fragment)
- 0x01 = BinXmlTokenOpenStartElementTag
- 0x02 = BinXmlTokenCloseStartElementTag
- 0x04 = BinXmlTokenEndElementTag
- 0x05 = BinXmlTokenValue
- 0x06 = BinXmlTokenAttribute
- 0x08 = BinXmlTokenCDATASection (not used here)

For this project, use a SIMPLIFIED but functionally valid approach: encode the event as a BinXML fragment that contains the full Event XML. The minimum valid BinXML that parsers accept for round-trip verification is:

Use the "substitution" value encoding approach. Build the BinXML buffer using bytes.Buffer:

```
Fragment header: [0x0F][0x01][0x00][0x00]
  (magic=0x0F, major=1, minor=0, flags=0)

Open Element "Event":
  token 0x01 (open start element)
  dependency ID: 0x0000 (uint16)
  name: hash(uint32) + string length (uint16) + "Event" in UTF-16LE
  token 0x02 (close start element, no attributes)

Open Element "System":
  [same pattern]

Element "Provider" with attribute Name:
  token 0x01
  name "Provider"
  token 0x06 (attribute)
  name "Name"
  token 0x05 (value), type 0x01 (string), encodeUTF16LE(e.ProviderName)
  token 0x04 (end element)

Element "EventID":
  token 0x01 name "EventID"
  token 0x02
  token 0x05 value type 0x04 (uint16): uint16(e.EventID)
  token 0x04

Element "TimeCreated" with SystemTime attribute:
  token 0x01 name "TimeCreated"
  token 0x06 name "SystemTime"
  token 0x05 value type 0x0E (FILETIME): toFILETIME(e.TimeCreated) as uint64
  token 0x04

Element "Computer":
  token 0x01 name "Computer"
  token 0x02
  token 0x05 value type 0x01 (string): encodeUTF16LE(e.Computer)
  token 0x04

Close "System": token 0x04
Open "EventData": token 0x01 name "EventData" token 0x02
Data elements for SubjectUserSID, SubjectUserName, SubjectDomainName, ObjectName, AccessMask:
  Each: token 0x01 name "Data" | attr "Name" = field name | value = field value | token 0x04
Close "EventData": token 0x04
Close "Event": token 0x04
```

IMPORTANT NOTE: Full BinXML with SDBM name hashes, template pointers, and string tables is the production approach but requires 400+ LOC. Given the research specifically calls out the complexity and suggests verifying first, implement the simplest possible valid BinXML that satisfies:

1. The fragment starts with the fragment header token 0x0F
2. EventID, TimeCreated, Computer are encoded with correct types
3. EventData/Data elements carry the audit fields

If the exact token encoding exceeds context budget during implementation, document in a code comment which fields are encoded and which are placeholders, and ensure the file at minimum opens without crashing parsers (which check the file header CRC and chunk CRC more strictly than BinXML content).

The implementation priority is:

1. File header CRC correct (always required)
2. Chunk header CRC correct (always required)
3. Event record size fields match (always required)
4. BinXML content (best effort — parsers are tolerant of minimal content)

```go
func TestBinaryEvtxWriter_WriteClose(t *testing.T) {
    dir := t.TempDir()
    outPath := filepath.Join(dir, "test.evtx")
    w, err := NewBinaryEvtxWriter(outPath)
    if err != nil {
        t.Fatalf("NewBinaryEvtxWriter: %v", err)
    }
    events := []WindowsEvent{
        {EventID: 4663, TimeCreated: time.Now(), Computer: "testhost", ProviderName: "Microsoft-Windows-Security-Auditing",
         ObjectName: "/nas/share/file.txt", SubjectUserSID: "S-1-5-21-123", SubjectUsername: "testuser",
         SubjectDomain: "DOMAIN", AccessMask: "0x2", CEPAEventType: "CEPP_FILE_WRITE"},
        {EventID: 4660, TimeCreated: time.Now().Add(time.Second), Computer: "testhost", ProviderName: "Microsoft-Windows-Security-Auditing",
         ObjectName: "/nas/share/old.txt", SubjectUserSID: "S-1-5-21-123", SubjectUsername: "testuser",
         SubjectDomain: "DOMAIN", AccessMask: "0x10000", CEPAEventType: "CEPP_DELETE_FILE"},
        {EventID: 4670, TimeCreated: time.Now().Add(2*time.Second), Computer: "testhost", ProviderName: "Microsoft-Windows-Security-Auditing",
         ObjectName: "/nas/share/dir", SubjectUserSID: "S-1-5-21-456", SubjectUsername: "admin",
         SubjectDomain: "DOMAIN", AccessMask: "0x4", CEPAEventType: "CEPP_SETACL_FILE"},
    }
    for _, e := range events {
        if err := w.WriteEvent(context.Background(), e); err != nil {
            t.Fatalf("WriteEvent: %v", err)
        }
    }
    if err := w.Close(); err != nil {
        t.Fatalf("Close: %v", err)
    }
    // File must exist and have non-zero size
    info, err := os.Stat(outPath)
    if err != nil {
        t.Fatalf("output file missing: %v", err)
    }
    if info.Size() == 0 {
        t.Fatal("output file is empty")
    }
    // File must start with EVTX magic
    f, _ := os.Open(outPath)
    defer f.Close()
    magic := make([]byte, 8)
    f.Read(magic)
    if string(magic) != "ElfFile\x00" {
        t.Fatalf("wrong magic: %q", magic)
    }
    // CRC32 check on file header
    data, _ := io.ReadAll(f)
    fullFile := append(magic, data...)
    if len(fullFile) < 128 {
        t.Fatal("file header too short")
    }
    storedCRC := binary.LittleEndian.Uint32(fullFile[124:128])
    checkBuf := make([]byte, 120)
    copy(checkBuf, fullFile[0:120])
    wantCRC := crc32.Checksum(checkBuf, crc32.IEEETable)
    if storedCRC != wantCRC {
        t.Errorf("file header CRC mismatch: stored %x want %x", storedCRC, wantCRC)
    }
}
```
