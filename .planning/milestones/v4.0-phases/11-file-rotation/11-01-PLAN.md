Current go-evtx Writer state (Phase 10 v0.3.0):

  type RotationConfig struct { FlushIntervalSec int }

  type Writer struct {
    mu sync.Mutex; path string; records []byte; recordID uint64
    firstID uint64; f *os.File; chunkCount uint16
    cfg RotationConfig; done chan struct{}; wg sync.WaitGroup
  }

  backgroundLoop() runs flush ticker only (no rotation ticker yet).
  Goroutine start condition: cfg.FlushIntervalSec > 0

  Format constants: evtxFileHeaderSize=4096, evtxChunkSize=65536
  flushChunkLocked(): writes chunk at evtxFileHeaderSize + chunkCount*evtxChunkSize,
    increments chunkCount, patches file header, calls f.Sync(), resets records/firstID.
  tickFlushLocked(): writes partial chunk without incrementing chunkCount.

  Reader API: Open(path), ReadRecord() (*Record, error), ErrNoMoreRecords, Record.RecordID
